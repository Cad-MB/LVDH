Titre : Analyse de graphe

Description : L'auteur souhaite générer la page HTML de son livre.

Acteur : Auteur

Préconditions: Le livre doit exister

Séquence nominale :

SN1. l'auteur a effectué une modification concernant les sections ou les enchainements du livre
SN2. Le système modifie le graphe d'analyse.

Post-Conditions

Le système a modifié le graphe d'analyse.
